# 🌡️ IoT Temperature & Humidity Monitoring System

## 📌 Description
This project uses ESP32 and DHT22 sensor to monitor temperature and humidity in real-time. Data is sent to ThingSpeak cloud for visualization.

## 🚀 Features
- Real-time monitoring
- Cloud integration (ThingSpeak)
- Threshold alert system
- Remote access

## 🛠️ Components Used
- ESP32
- DHT22 Sensor
- Jumper Wires
